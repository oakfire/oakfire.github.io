About Processing.js

  1. Processing.js is the sister project of the Processing visual programming language, designed for the web.
  2. Processing.js makes your data visualizations, digital art, interactive animations, educational graphs, video games, etc.
  3. Allowing Processing code to be run by any HTML5 compatible browser

Using Processing

two ways of implementing processing

First Mehtod

Needed files:

  1. processing.js
  2. anything.html
  3. anything.pde

The anything.html file will look like:


              <script src="processing.js"></script>
              <canvas data-processing-sources="anything.pde"></canvas>

The anything.pde file will look like:


              void setup()
              {
                size(200,200);
                background(125);
                fill(255);
                noLoop();
                PFont fontA = loadFont("courier");
                textFont(fontA, 14);  
              }

              void draw(){  
                text("Hello Web!",20,20);
              }

Second Method

Needed files:

  1. processing.js
  2. anything.html

In second method the anything.html file will look like:


              <script src="processing.js"></script>
              <script type="text/processing" data-processing-target="mycanvas">
              void setup()
              {
                size(200,200);
                background(125);
                fill(255);
                noLoop();
                PFont fontA = loadFont("courier");
                textFont(fontA, 14);  
              }

              void draw(){  
                text("Hello Web!",20,20);
              }
              </script>
              <canvas id="mycanvas"></canvas>

Basic Syntax

A brief look at the structure of a Processing sketch reveals how easy it is to program interactive visualizations.

  1. Begin by defining global variables
  2. Create a setup() function, control the visualization's properties
  3. Create a draw() function, loops continuously
  4. Adding interactivity, such as mousePressed()

1.Begin by defining global variables


              // Global variables
              float radius = 50.0;
              int X, Y;
              int nX, nY;
              int delay = 16;

2.Create a setup() function, control the visualization's properties


              // Setup the Processing Canvas
              void setup(){
                size( 200, 200 );
                strokeWeight( 10 );
                frameRate( 15 );
                X = width / 2;
                Y = height / 2;
                nX = X;
                nY = Y;  
              }

3.Create a draw() function, loops continuously


              // Main draw loop
              void draw(){
                radius = radius + sin( frameCount / 4 );
                // Track circle to new destination
                X+=(nX-X)/delay;
                Y+=(nY-Y)/delay;
                // Fill canvas grey
                background( 100 );
                // Set fill-color to blue
                fill( 0, 121, 184 );
                // Set stroke-color white
                stroke(255); 
                // Draw circle
                ellipse( X, Y, radius, radius );                  
              }
